# 李昊冉 · 求职简历网站

## 📁 文件说明

| 文件 | 用途 |
|------|------|
| `index.html` | 简历网页，浏览器直接打开即可查看，A4 打印即简历 |
| `resume.md` | Markdown 源文件（备用） |

## ✏️ 如何修改简历内容

用**记事本**打开 `index.html`，搜索 `👇` 符号，每个 `👇` 下面一行就是你要改的内容：

- 名字 → 搜 `改名字`
- 电话 → 搜 `改电话`
- 学校 → 搜 `改学校名`
- 意向岗位 → 搜 `改意向岗位`
- 照片 → 搜 `照片替换成`

改完保存，刷新浏览器就能看到效果。

## 🌐 部署到 Gitee Pages（免费、国内访问快）

### 第一步：注册 Gitee

打开 https://gitee.com → 点右上角「注册」→ 用手机号注册账号。

> ⚠️ 注册后需要**实名认证**才能开通 Pages 服务。在「个人设置 → 安全设置 → 实名认证」里完成。

### 第二步：创建仓库

1. 登录后点右上角 **+** →「新建仓库」
2. 仓库名称填 `resume`（或你喜欢的名字）
3. 路径会自动生成
4. 公开/私有都可以（Pages 都支持）
5. 不要勾选「初始化仓库」
6. 点「创建」

### 第三步：上传文件

创建完仓库后会看到「上传文件」的提示。把以下两个文件拖进去：
- `index.html`
- `resume.md`（可选）

或者直接用 Git 推送：
```bash
git init
git add index.html resume.md
git commit -m "简历第一版"
git remote add origin https://gitee.com/你的用户名/resume.git
git push -u origin master
```

### 第四步：开启 Gitee Pages

1. 进入仓库 → 顶部菜单点「服务」→「Gitee Pages」
2. 部署分支选 `master`
3. 部署目录留空（默认根目录）
4. 点「启动」

等几秒钟，会生成一个网址，格式为：
```
https://你的用户名.gitee.io/resume
```

### 第五步：发给面试官

把上面这个网址直接微信发给面试官，对方在手机上点开就能看。

> 💡 每次修改 `index.html` 后，需要回到 Gitee Pages 页面点一下「更新」按钮，网站才会刷新。

## 🖨️ 打印成纸质简历

1. 用浏览器打开 `index.html`
2. 按 `Ctrl + P`
3. 纸张大小选 **A4**
4. 边距选「无」
5. 勾选「背景图形」
6. 打印即可
