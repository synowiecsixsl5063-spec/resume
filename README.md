## ✏️ 如何修改内容

网站上线后，每次改内容只需两步：

1. 用**记事本**打开 `index.html`，搜 `👇` 找到要改的地方
2. 改完保存，在终端里运行：

```bash
cd resume-project
git add index.html && git commit -m "更新简历" && git push
```

推送后等约 20 秒，刷新网页即可看到更新。

---

## 📍 需要改的地方一览

| 搜这个 | 修改内容 |
|--------|----------|
| `改名字` | 姓名 |
| `改电话` | 电话、现居地、邮箱 |
| `改学校名` | 学校、专业、时间 |
| `改意向岗位` | 求职意向 |
| `照片替换成` | 替换照片框为真实证件照 |
